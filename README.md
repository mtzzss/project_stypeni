 # Project Stypeni

Простая программа для анализа местности на спутниковых снимках. Она находит зоны (лес, вода, песок), считает их площадь и показывает изменения территории со временем.

## Возможности

* **Выделение зон:** Автоматически находит лес, воду и другие объекты по цвету.
* **Расчет площади:** Показывает размеры выделенных участков в квадратных метрах.
* **Графики:** Строит историю изменений по датам.
* **Сравнение:** Находит отличия между старым и новым снимком (что исчезло, что появилось).

## Установка и запуск

Вам понадобится Python. Выполните следующие команды в терминале:

**1. Установите библиотеки:**
```bash
pip install streamlit opencv-python-headless numpy pandas pillow
```
**2. Запустите приложение:**
```bash
streamlit run main.py
```

## Как пользоваться
* **Вход:** При первом запуске зарегистрируйте новый аккаунт.

* **Новый анализ:** Загрузите фото, выберите тип местности, введите название территории и нажмите "Сохранить в Архив".

* **Динамика:** Перейдите на вкладку "Динамика", чтобы увидеть график изменений площади.

* **Сравнение:** Перейдите на вкладку "Сравнение", выберите два снимка, и программа покажет разницу.
