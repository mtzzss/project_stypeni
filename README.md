 Stypeni

Простая программа для анализа местности. Она находит зоны (лес, вода, песок) на фото, считает их площадь и показывает, как менялась территория со временем.

Что умеет

 Выделять зоны: Красит лес, воду и другие объекты.

 Считать площадь: Показывает размеры в м².

 Строить графики: Показывает историю изменений.

 Сравнивать: Находит отличия между старым и новым снимком.

Как запустить

Откройте терминал в папке с проектом и введите:

1. Установка библиотек:
```
pip install streamlit opencv-python-headless numpy pandas pillow
```

2. Запуск:
```
streamlit run a.py
```

Как пользоваться

Вход: Придумайте логин и пароль.

Новый анализ: Загрузите фото, выберите тип местности и нажмите "Сохранить в Архив".

Динамика: Посмотрите график изменений площади.

Сравнение: Выберите два снимка, чтобы увидеть, что исчезло или появилось.
