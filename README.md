Как пользоваться приложением
Предварительные требования
1) Установленный Python 3.8 или выше
2)Менеджер пакетов pip.

Установка и запуск:

1.Клонируйте репозиторий:
bash
git clone https://github.com/mtzzss/project_stypeni.git
cd project_stypeni

2.Установите необходимые библиотеки:
bash
pip install -r requirements.txt

3.Запустите приложение:
bash
streamlit run app.py

Приложение автоматически откроется в вашем браузере по адресу http://localhost:8501.
------------------------------------------------------------------------------------
Работа с приложением интуитивна и состоит из нескольких шагов:

Загрузка: Перетащите или выберите файл спутникового снимка или карты (поддерживаются JPG, PNG).

Выбор категорий: В боковой панели отметьте интересующие категории (Вода, Лес, Горы и др.).

Выбор подзон: Для каждой категории уточните, какие конкретно типы объектов анализировать (например, в категории "Вода" можно выбрать "река" и "озеро").

Настройка масштаба: Укажите, сколько квадратных метров соответствует одному пикселю на вашем изображении. Это ключевой параметр для пересчета площади.

Анализ: Приложение автоматически обработает изображение и отобразит результаты в трех колонках: исходное фото, цветная маска выделенных областей и контуры объектов.

Итоги: Внизу страницы появится подробная таблица с расчетом площади для каждого типа зоны, а также общая статистика.
